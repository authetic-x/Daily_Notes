## 三列布局

### 基于浮动

```js
<div class="wrapper">
    <div class="left"></div>
    <div class="right"></div>
	<div class="content"></div>
</div>

.left {
    height: 100%;
    float: left;
    width: 200px;
}
.right {
    height: 100%;
    float: right;
    width: 200px;
}
.content {
    height: 100%;
    margin: 0 200px;
}
```

### 基于绝对定位(用的相对较少，高度未知会出现问题)

```js
<div class="wrapper">
    <div class="left"></div>
	<div class="content"></div>
    <div class="right"></div>
</div>

.wrapper {
    position: relative;
}
.left {
    position: absolute;
    left: 0;
    width: 200px;
    height: 100%;
}
.right {
    position: absolute;
    right: 0;
    width: 200px;
    height: 100%;
}
.content {
    margin: 0 200px;
    height: 100%;
}
```

### 基于flex

```js
<div class="wrapper">
    <div class="left"></div>
	<div class="content"></div>
    <div class="right"></div>
</div>

.wrapper {
    display: flex;
}
.left .right {
    width: 200px;
}
.content {
    flex: 1;
}
```
